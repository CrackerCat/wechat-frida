wechat-frida
========================================

介绍
-----------------------------------------
* wechat-frida 是一款使用frida框架hook微信PC端的聊天机器人框架。该框架使用frida js脚本动态hook程序，便于调试和快速适配最新客户端。
* 仓库地址：https://github.com/luoyeah/wechat-frida
* 涉及技术：二进制逆向分析、frida动态hook、python、fastapi
