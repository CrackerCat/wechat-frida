### 发送一条文本消息

#### 接口说明

> 发送一条文本消息

#### 请求方式

> POST

#### 请求地址

> /send_message

#### 返回格式

> JSON

#### 请求参数

| 参数      | 必选   | 类型     | 默认值  | 说明|
|---------|-------|-------|-------|-------|
| wxid    | √    | string |      | wxid。 |
| message | √    | string |      | 消息。 |

#### 返回字段

| 返回字段                  | 字段类型 | 说明     |
|-------------------------|--------|--------|
| msg                     | string | 响应消息。  |
| code                    | int    | 响应代码。  |

#### 字段变化

无

#### 接口示例

> 发送示例

```javascript
{
    "wxid": "filehelper",
    "message" : "你好"
}
```

> 响应示例

```javascript
{
	"msg": "",
	"code": 0
}
```
